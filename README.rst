
=========================
Leonardo Stavbyaz Theme
=========================

Leonardo theme for Stavby od A po Z

Installation
============

.. code-block:: bash

    pip install leonardo-theme-stavbyaz

After installation don't forget run ``sync_all`` command.

.. code-block:: python

    python manage.py sync_all
